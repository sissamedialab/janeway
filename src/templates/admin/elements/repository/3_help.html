• Submission Start Text
    • This text is shown at the top of the submission start page.
  • File Upload Help
    • Provided additional text to guide authors during the file upload process.
  • Limit Upload to PDF
    • Check this box if you want to limit authors to upload only PDFs.
  • Limit Upload to PDF Help
    • Combines with the repository name and displays on the file upload page. With the default text it will read "RepositoryName requires that all author uploads be PDF files".
  • Managers
    • You can select accounts that you want to be managers of this repository. They can access the review submissions.
